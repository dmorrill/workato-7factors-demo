# Workato Developer Marketing Demo

A Laravel app that serves two purposes:
1. **Seven Factors site** — a Wolf2-branded developer landing page explaining the Seven Factors of Enterprise-Ready Agentic Integration
2. **Campaign Maker** — an internal tool that turns any URL (web page or YouTube video) into a landing page + full campaign brief, generated by Claude

Built for Workato's developer marketing team to demonstrate content amplification.

---

## What it does

### Seven Factors site (`/`)

Static-ish marketing site explaining Workato's Seven Factors framework. Styled per Wolf2 brand guidelines (flat navy hero, teal `#67EADD` accent, Besley display font, Nunito Sans body, JetBrains Mono for code).

Routes:
- `/` — landing page
- `/campaign` — campaign brief (static)
- `/factor/1` through `/factor/7` — individual factor pages

### Campaign Maker (`/make`)

Internal tool for marketers. Give it a URL, it generates a developer-facing landing page and campaign brief using Claude.

**Intake:** marketer provides a campaign name (optional), one or more source URLs, and selects which developer personas to target.

**Generation:** Claude makes two API calls — one for landing page content, one per persona for 6 campaign packages (blog, LinkedIn, Twitter/X, newsletter, video script, developer tutorial).

**Review:** marketer reviews the generated content, approves or skips individual packages, toggles between persona views, then publishes.

**Output:** two clean public URLs at `/generated/{slug}` and `/generated/{slug}/campaign`.

## Developer personas

Packages are generated per persona, with different emphasis, proof points, and tone:

| Key | Label | Group |
|-----|-------|-------|
| `enterprise_automation` | Enterprise Automation Engineer | Production Builders |
| `agent_architect` | Agent Systems Architect | Production Builders |
| `product_ai_builder` | Product-Facing AI Builder | Production Builders |
| `trust_risk` | Trust & Risk Steward | Trust & Risk Stewards |
| `ai_learner` | AI Learner & Accelerator | AI Learners & Accelerators |

Persona definitions (messaging context, emphasis, tone) live in `CampaignGeneratorService::personaDefinitions()`.

## Tech stack

- **Laravel 13** + **SQLite** (default, no config needed)
- **Tailwind CSS v4** via `@tailwindcss/vite` — token config in `resources/css/app.css` using `@theme` block
- **Alpine.js v3** — tabs, modals, loading overlays, dynamic URL list
- **Vite** for asset bundling
- **Anthropic API** (Claude Opus 4.6) — called directly via Laravel HTTP facade, no package

## Local setup

```bash
git clone https://github.com/dmorrill/workato-7factors-demo.git
cd workato-7factors-demo

composer install
npm install

cp .env.example .env
php artisan key:generate
```

Add your Anthropic API key to `.env`:
```
ANTHROPIC_API_KEY=sk-ant-...
```

```bash
php artisan migrate
npm run dev        # in one terminal
php artisan serve  # in another
```

App runs at `http://localhost:8000`. Campaign Maker at `/make`.

## Database schema

**campaigns**
- `slug`, `name` (user label), `title` (generated), `status` (draft/published)
- `source_url` (primary), `source_urls` (JSON array, all URLs)
- `source_type` (webpage/youtube), `video_url`
- `hero_headline`, `hero_subheadline`, `framework_name`, `framework_intro`
- `context_headline`, `context_body`, `cta_headline`, `cta_body`
- `extracted_content` (longText), `personas` (JSON array of selected persona keys)

**campaign_concepts** — `campaign_id`, `sort_order`, `number`, `name`, `principle`

**campaign_packages** — `campaign_id`, `channel`, `title`, `copy`, `status` (pending/approved/skipped), `persona`

## Wolf2 brand rules

- Hero backgrounds: flat `#083763` navy — no gradients, no textures
- Teal `#67EADD` is a **light** color — always use `text-[#083763]` on teal backgrounds
- Sand sections: `bg-[#F4F2E3]`
- Teal accent in content only (borders, pills, highlights) — never as a large background
- Display font: Besley (`font-display`), Body: Nunito Sans, Code: JetBrains Mono (`font-mono-code`)
- Full brand reference: `workato-consulting` repo → `intel/brand-guidelines.md`
